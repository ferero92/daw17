Para poder poner a punto la aplicación debemos iniciar sesión como Administrador utilizando los
siguientes datos:

	Usuario -> secretario@ejemplo.com
	Password -> 123

Una vez accedamos al panel de control, debemos entrar en "modificar usuario" para establecer un
correo electrónico válido tanto para el secretario como para el encargado de mantenimiento.

Finalmente, debemos crear un profesor mediante la opción del panel de control "crear nuevo usuario"
y ya lo tendremos todo a punto para iniciar nuestra aplicación.

**Los emails serán enviados a través de una cuenta de correo llamada: "mailgeneric5@gmail.com".
Los datos de esta cuenta de correo pueden cambiarse en la clase model.From**

***La aplicación ha sido creada siguiendo el patrón de diseño de Modelo Vista Controlador, siendo
el modelo las clases incluidas en le paquete "model", la vista los distintos archivos JSP incluidos
en la carpeta "pages", y el controlador los Servlets que se encuentran en el paquete "controller"***
