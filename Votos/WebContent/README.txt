  --  Introducción y despliegue  --
Para poder utilizar esta aplicación no necesitaremos de ningún usuario o login, por lo tanto sólo
debemos desplegarla en nuestro servidor y acceder a localhost/Votos para comenzar.

  --  Acciones del administrador  --
La página index de la aplicación es un panel que nos permite añadir un nuevo candidato de una
categoría especificada. Una vez añadido, podemos consultar en la pestaña listado todos los
candidatos con los que cuenta el sistema, así como unos parámetros para filtrar. La aplicación
recoge automáticamente el curso en el que se va a realizar la votación. Si añadimos un candidato
antes de septiembre de este año, este pertenecerá al curso 2016/2017, mientras que si lo añadimos
en dicho mes, permanecerá al curso siguiente (2017/2018). No debe preocuparse en añadir
candidatos, pues la aplicación ya incluye varios para poder probarla. También, si lo desea, puede
eliminar todos los datos de votaciones anteriores desde la opción de "Borrar curso seleccionado".

  --  Pestaña Votar  --
La siguiente pestaña es la de Votar. En ella llevaremos a cabo la votación marcando tantos
candidatos como deseemos. Una vez marcados nos aparecerá una ventana emergente para confirmar
nuestra selección y en una pestaña nueva del navegador accederemos a la venta de impresión para
tener nuestro voto de manera física.

  --  Consulta de resultados  --
El último apartado es el Escrutinio. Como puede imaginarse, en esta pestaña veremos el total de
votos emitidos así como el porcentaje de manera gráfica de cada candidato. La aplicación cuenta con
un botón para reiniciar los votos a cero de todos los candidatos de una categoría determinada.
